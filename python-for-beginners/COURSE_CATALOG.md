# Course Catalog

15 modules, roughly one module per week at 1.5–2 hours/day, 6 days/week.
Check off each module as you complete it.

| #  | Module Name                  | Focus                                              | Status |

| 01 | Getting Started              | Python install, REPL, print(), comments            | ☐      |

| 03 | Strings                      | Literals, f-strings, indexing, slicing, methods    | ☐      |
| 04 | Operators & Expressions      | Arithmetic, comparison, logical, precedence        | ☐      |
| 05 | Control Flow                 | if/elif/else, truthiness, nested, match            | ☐      |
| 06 | Loops                        | for, range(), while, break, continue, nested       | ☐      |
| 07 | Lists & Tuples               | Create, slice, methods, comprehensions, tuples     | ☐      |
| 08 | Dictionaries & Sets          | dicts, keys/values, nested dicts, sets, operations | ☐      |
| 09 | Functions                    | def, parameters, return, scope, docstrings         | ☐      |
| 10 | File I/O                     | open(), read/write, with, csv, pathlib             | ☐      |
| 11 | Error Handling               | try/except/else/finally, raise, common exceptions  | ☐      |
| 12 | Modules & Packages           | import, stdlib, pip, venv, writing your own module | ☐      |

## How to mark a module complete

Open this file, change the `☐` to `☑` for the module you finished, save the file.
