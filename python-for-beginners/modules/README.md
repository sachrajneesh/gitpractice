Each module folder contains:
- `README.md` — objectives, topics covered, exercise checklist
- `CONCEPTS.md` — teaching content with examples
- `examples/` — runnable Python files demonstrating each concept
- `exercises/` — starter files (with TODOs) and solution files

Work through modules in order. Do not skip ahead.
