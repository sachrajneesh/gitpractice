## Learning objectives

By the end of this module you will be able to:

- Run a Python program three different ways (REPL, script file, VS Code)
- Use `print()` to display text, numbers, and multiple values
- Control `print()` output with `sep` and `end` parameters
- Write comments with `#` and explain why they matter
- Read a Python error message and identify what went wrong

---

## Topics covered

1. What is a program? (plain English)
2. What is Python and why learn it?
3. Three ways to run Python: REPL, script file, VS Code
4. `print()` — the most-used function in Python
5. `sep` and `end` parameters
6. Comments (`#`) — what they are and when to use them
7. Common beginner errors: SyntaxError and NameError
