## Learning objectives

By the end of this module you will be able to:

- Create variables of type int, float, str, bool, and NoneType
- Explain the "labelled box" analogy for variables
- Name variables correctly using snake_case conventions
- Use `type()` to check what type a value is
- Use `input()` to get text from the user
- Convert user input to a number with `int()` and `float()`

---

## Topics covered

1. What is a variable? (the labelled box analogy)
2. The five basic data types: int, float, str, bool, None
3. Variable naming rules: snake_case, no spaces, no leading digit
4. The `type()` function — checking what type a value is
5. `input()` — getting text from the user
6. Type conversion — `int()`, `float()`, `str()`
7. Common type gotcha: adding a number to a string
