# Module 04: Operators & Expressions

> **30/70 reminder** — 30% reading, 70% coding. After CONCEPTS.md, go straight
> to the exercise. Experiment freely in the REPL to test operator behaviour.

---

## Learning objectives

By the end of this module you will be able to:

- Use all arithmetic operators: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- Use comparison operators: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Use logical operators: `and`, `or`, `not`
- Understand operator precedence and use parentheses to control it
- Build a simple command-line calculator

---

## Topics covered

1. Arithmetic operators and what they return
2. Integer vs float division (`/` vs `//`)
3. The modulo operator `%` and its practical uses
4. Exponentiation with `**`
5. Comparison operators (return `True` or `False`)
6. Logical operators (`and`, `or`, `not`)
7. Operator precedence — PEMDAS / BODMAS for Python
8. Augmented assignment: `+=`, `-=`, `*=`, `/=`
