# Module 05: Control Flow

> **30/70 reminder** — Control flow is all about making decisions. The only way
> to build intuition for it is to write lots of if/elif/else chains yourself.
> Keep the reading short and the coding long.

---

## Learning objectives

By the end of this module you will be able to:

- Write `if`, `elif`, and `else` statements
- Understand **truthiness** — what values Python treats as True or False
- Write nested conditions
- Use Python's `match` statement (Python 3.10+) for clean multi-branch logic

---

## Topics covered

1. `if` / `elif` / `else` — the basic decision structure
2. Truthiness and falsy values (`0`, `""`, `None`, `[]`, `{}`)
3. Nested conditions — if inside if
4. The `match` statement — Python's pattern matching (3.10+)
5. Writing clean conditions with logical operators
