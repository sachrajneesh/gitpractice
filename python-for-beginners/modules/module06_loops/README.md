# Module 06: Loops

> **30/70 reminder** — Loops are best learned by writing them, not reading about
> them. After CONCEPTS.md, write every example yourself and modify each one.

---

## Learning objectives

By the end of this module you will be able to:

- Use a `for` loop to iterate over a range or a sequence
- Use `range()` with start, stop, and step arguments
- Use a `while` loop for "keep going until" logic
- Use `break` to exit a loop early
- Use `continue` to skip to the next iteration
- Write simple nested loops

---

## Topics covered

1. `for` loop — iterating over a sequence
2. `range(start, stop, step)` — generating number sequences
3. `while` loop — condition-controlled repetition
4. `break` — exit the loop early
5. `continue` — skip the rest of this iteration
6. `else` on loops — runs if loop completed without `break`
7. Nested loops — a loop inside a loop
