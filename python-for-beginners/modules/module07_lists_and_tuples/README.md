# Module 07: Lists & Tuples

> **30/70 reminder** — Lists are Python's workhorse data structure. You will use
> them constantly. Read CONCEPTS.md once, then spend the bulk of your time
> building the to-do list app in the exercise.

---

## Learning objectives

By the end of this module you will be able to:

- Create, access, and modify lists
- Use list methods: `append`, `extend`, `insert`, `remove`, `pop`, `sort`, `reverse`
- Slice lists just like strings
- Write list comprehensions
- Explain the difference between lists (mutable) and tuples (immutable)

---

## Topics covered

1. Creating lists and accessing elements by index
2. Negative indexing and slicing
3. Common list methods
4. Iterating over a list with a `for` loop
5. List comprehensions — the Pythonic shorthand for filtered/transformed lists
6. Tuples — immutable sequences, when to use them
7. Tuple unpacking
