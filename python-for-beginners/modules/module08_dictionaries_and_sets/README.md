# Module 08: Dictionaries & Sets

> **30/70 reminder** — Dictionaries are everywhere in real Python code.
> The exercise is a contacts book — a program you might actually use.

---

## Learning objectives

By the end of this module you will be able to:

- Create and use dictionaries with key-value pairs
- Get, set, and delete values
- Iterate over keys, values, and items
- Build nested dictionaries
- Create sets and perform union, intersection, and difference operations

---

## Topics covered

1. Dictionaries — key-value stores
2. Accessing values: `dict[key]` vs `dict.get(key, default)`
3. Setting, updating, and deleting keys
4. Iterating: `.keys()`, `.values()`, `.items()`
5. Nested dictionaries
6. Sets — unordered collections of unique values
7. Set operations: union (`|`), intersection (`&`), difference (`-`)
