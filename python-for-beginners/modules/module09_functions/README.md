# Module 09: Functions

> **30/70 reminder** — Functions are the single most important concept in
> programming. Do not rush CONCEPTS.md, but make sure 70% of your time is
> spent writing functions, not reading about them.

---

## Learning objectives

By the end of this module you will be able to:

- Define a function with `def` and call it
- Use parameters and return values
- Write default parameter values
- Use keyword arguments when calling a function
- Explain scope: what is local vs global
- Write a proper docstring for a function

---

## Topics covered

1. Defining and calling functions
2. Parameters (inputs) and return values (outputs)
3. Default parameter values
4. Keyword arguments vs positional arguments
5. Variable scope — local vs global
6. Docstrings — documenting your functions
7. When to use functions (DRY principle)
