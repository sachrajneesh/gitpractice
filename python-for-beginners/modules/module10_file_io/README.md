# Module 10: File I/O

> **30/70 reminder** — File I/O is a practical skill. The best way to learn it
> is to write programs that actually create and read files on your own computer.
> Spend 70% of your time in the exercise writing real files.

---

## Learning objectives

By the end of this module you will be able to:

- Open, read, and write text files with `open()` and the `with` statement
- Read files line by line
- Append to existing files
- Read and write CSV files using the `csv` module
- Use `pathlib.Path` for clean file path handling

---

## Topics covered

1. `open()` and file modes: `"r"`, `"w"`, `"a"`, `"x"`
2. `read()`, `readlines()`, `readline()`
3. `write()` and `writelines()`
4. The `with` statement — why it matters (auto-close)
5. Reading files line by line with a `for` loop
6. The `csv` module — `csv.reader` and `csv.writer`
7. `pathlib.Path` — modern file path handling
