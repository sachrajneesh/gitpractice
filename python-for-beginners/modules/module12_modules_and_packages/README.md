# Module 12: Modules & Packages

> **30/70 reminder** — The standard library is enormous. Do not try to memorise
> it. Learn the four or five most useful modules thoroughly, then look up the rest
> as you need them. The exercise gets you coding with real stdlib modules.

---

## Learning objectives

By the end of this module you will be able to:

- Use `import` and `from X import Y` correctly
- Use key standard library modules: `math`, `random`, `datetime`, `os`, `sys`
- Install third-party packages with `pip`
- Create and use a virtual environment (`venv`)
- Write your own module and import it into another file

---

## Topics covered

1. `import` statement — full module import
2. `from X import Y` — importing specific names
3. Standard library tour: `math`, `random`, `datetime`, `os`, `sys`
4. `pip install` and `requirements.txt`
5. Creating and activating a virtual environment with `venv`
6. Writing your own module (a `.py` file with functions)
7. `if __name__ == "__main__":` — running a file as a script vs importing it
