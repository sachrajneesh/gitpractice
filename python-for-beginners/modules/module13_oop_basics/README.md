# Module 13: OOP Basics

> **30/70 reminder** — Object-oriented programming is a different way of thinking
> about code. It takes time to click. Read CONCEPTS.md slowly, then build the
> BankAccount class step by step. Do not skip the inheritance part.

---

## Learning objectives

By the end of this module you will be able to:

- Define a class with `__init__` and instance attributes
- Write instance methods that operate on `self`
- Implement `__str__` for readable string representation
- Create a subclass using inheritance
- Call the parent class constructor with `super()`

---

## Topics covered

1. What is a class? (blueprint analogy)
2. `__init__` — the constructor method
3. Instance attributes (`self.name`, etc.)
4. Instance methods
5. `__str__` — how your object prints itself
6. Inheritance — one class built on top of another
7. `super()` — calling the parent class
